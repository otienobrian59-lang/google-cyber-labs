# Module 1: Foundations of Cybersecurity

**Status**: Completed ✅

## 🌟 Course Overview
This course provides an introduction to the field of cybersecurity, covering its history, core concepts, and the roles of cybersecurity professionals.

---

## 🔑 Key Concepts Learned

### 1. The CIA Triad
The foundation of cybersecurity, used to guide security policies and identify vulnerabilities.
- **Confidentiality**: Ensuring sensitive information is accessed only by authorized individuals.
- **Integrity**: Ensuring data is accurate, complete, and has not been tampered with.
- **Availability**: Ensuring information and systems are accessible when needed.

### 2. Security Frameworks & Controls
- **NIST Cybersecurity Framework (CSF)**: A voluntary framework consisting of standards, guidelines, and best practices to manage cybersecurity-related risk.
  - *Identify, Protect, Detect, Respond, Recover*
- **CIS Controls**: A prioritized set of actions to protect organizations and data from known cyber attack vectors.
- **Security Controls**:
  - **Technical**: Firewalls, encryption, authentication.
  - **Administrative**: Policies, training, risk assessments.
  - **Physical**: Locks, security cameras, badges.

### 3. CISSP 8 Domains
1. Security and Risk Management
2. Asset Security
3. Security Architecture and Engineering
4. Communication and Network Security
5. Identity and Access Management (IAM)
6. Security Assessment and Testing
7. Security Operations
8. Software Development Security

### 4. Evolution of Cybersecurity
- **Brain Virus (1986)**: The first IBM PC compatible virus.
- **Morris Worm (1988)**: One of the first worms distributed via the Internet; led to the creation of the first CERT.
- **Modern Era**: Shift towards sophisticated state-sponsored attacks, ransomware, and cloud security.

### 5. Cybersecurity Roles & Skills
- **Roles**: SOC Analyst, Incident Responder, Penetration Tester, Security Engineer.
- **Skills**: Problem-solving, attention to detail, technical proficiency (Linux, SQL, Python), and communication.

---

## 🛠️ Tools & Programming
- **Operating Systems**: Introduction to Linux and its importance in security.
- **Programming**: Overview of how Python and SQL are used by analysts.
- **Ethics**: The importance of "white hat" hacking and professional integrity.

---

## 📝 Important Takeaways
- Cybersecurity is a continuous process, not a one-time setup.
- Understanding the "why" (business impact) is as important as the "how" (technical execution).
- Security is everyone's responsibility within an organization.
